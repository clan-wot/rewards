Чтобы настроить данные пользователя в базе автоматически, вы можете использовать браузер Google Chrome и установить специальное расширение.

У тех пользователей в базе, у которых прописаны их форумные идентификаторы, их ники являются ссылками, при клике по которым можно будет перейти на страницу их профиля на форуме. После установки расширения вы можете сделать правый клик над ссылкой-ником пользователя, из контекстного меню выбрать пункт "Загрузить с форума". Данные этого пользователя в таблице будут установлены согласно его текущим настройкам на форуме (вы должны быть залогинены в браузере на форуме).
